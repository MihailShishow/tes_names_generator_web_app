Welcome to the Names Generator!

Hello, this application was initially designed as an attempt to create a tool for names generation for The Elder Scrolls games universe. In the future, more settings and universes may be added.

By using slightly modified code from here, this application helps to generate names for the characters of all of the playable races from The Elder Scrolls games, starting from the third chapter.

Base data was taken from the UESP, then normalized and processed to generate new names.

In order to begin, please click here, and you'll be taken to the names generator interface.

Technical Information

This application was built using the following stack of technologies:

  • Flask
  • nginx
  • Docker
  • Bootstrap 4
  • Darkly theme for Bootstrap
